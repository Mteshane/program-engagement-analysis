# Program Engagement Analysis (Excel)

## Overview
This project presents an Excel-based analysis of user engagement data
across multiple digital programmes. The work focuses on transforming
raw user activity data into actionable insights using pivot tables
and an interactive dashboard.

## What This Project Demonstrates
- Data cleaning and preparation in Excel
- Pivot table creation for analytical insights
- Dashboard design for decision-making
- Interpretation of user behaviour data
- Clear reporting and attention to detail

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard)
- GitHub (documentation and version control)

## Files
- `Intern_Assignment_User_Activity_Dashboard.xlsx` – Excel analysis and dashboard
- `dashboard.png` – Screenshot of the final dashboard
